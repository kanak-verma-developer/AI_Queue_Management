AI Smart Queue Management System  
Internship Project – Kanak Verma

Real-time AI based queue system using YOLOv8 and Streamlit.

## 🔥 Features
- People detection from image
- Live camera detection
- Priority token system
- Manual entry option
- Queue analytics dashboard
- CSV + PDF export

## 🛠 Tech Stack
Python, OpenCV, YOLOv8, Streamlit, Pandas

## ▶ Run Project
streamlit run dashboard/app.py

Note/-models/ not included due to large file size
